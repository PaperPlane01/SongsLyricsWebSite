1. Реализовать регистрацию пользователей.+
2. Реализовать возможность логиниться/разлогиниться.+
3. Реализовать разделение пользователя по ролям: обычный пользователь, админ.+
4. Сделать удобный редактор текста песни, данные их которого можно было бы удобно считывать для дальнейшей работы с ними (например, добавление песни в базу данных и тд).+
5. Дать пользователям возможность предлагать тексты песен на рассмотрение.+
6. Дать админам возможность одобрять песни (текст песни становится доступным для всех)+, либо отклонять (текст песни удаляется из базы).
7. Дать админам возможность самостоятельно добавлять тексты песен в базу, пользуясь удобным редактором из п. 4 (песни, добавленные админом, автоматически становятся одобренными).
8. Реализовать возможность оценивать песни по 5-бальной шкале.
9. Реализовать поиск песен по различным параметрам.
9.1 По исполнителю
9.2 По названию песни
9.3 По жанрам
9.4 По всем вышеперечисленным параметрам
10. Сортировка выборки песен должна осуществляться по следующим критериям в прямом и обратном порядке:
10.1 По алфавиту
10.2 По рейтингу
10.3 По дате добавления
11. Реализовать возможность посмотреть список песен с наиболее высоким рейтингом (5-10 песен).
12. Реализовать возможность посмотреть список недавно добавленных песен (5-10 песен).
13. Реализовать возможность комментировать песни.
14. Дать админам возможность удалять комментарии и банить пользователей.
15. Сделать нормальный дизайн.
16. Доработать проект в соответствии с требованиями.
